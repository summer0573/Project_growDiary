## 소개
자신의 반려식물을 키우면서 성장과정을 기록하는 앱

## 기능
00 firebase을 사용하여 식물 정보 저장 <br>
01 실행했을때 화면 <br>
02 + 버튼을 누른뒤의 화면 <br>
03 식물 종류를 선택 <br>
04 날짜 선택 <br>
